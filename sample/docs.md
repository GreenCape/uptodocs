# UpToDocs

Try UpToDocs on this Markdown file.

## Examples

This is how you do multiplication:

```php
<?php
$v = multiply(1,2);
```

And another, but with a typo:

```php
<?php
$v = multiplyy(10,2);
```

Another wrong one:

```php
<?php
$v = divide(10,2);
```

Another good one:

```php
<?php
$v = multiply(3,4);
assert($v == 12);
```
